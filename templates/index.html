AI Bedtime Story Generator

Afrika Tales(July 2023)

Open AI API, Python (Flask), HTML & CSS

Representation isn't only important in Hollywood movies. It starts as early as the bedtime stories you read as a child. Afrika Tales uses the Open AI API to generate custom stories that African parents can read to their children.

Functionality: photo and text generation, samples shown on the site, and instructions for people to run the program. In the next iteration, users will be able to generate stories on the site.

Police Shooting: Data Analysis

Police Shooting Analysis(Aug 2023)

Python (Pandas, NumPy, Seaborn & other data viz modules)

What shapes your perception of police brutality? Catchy but biased news headlines? I dive into five datasets from the Washington Post to depict police shootings in the US in a more nuanced manner. I use bias mitigation strategies throughout to increase realiability.

Features: a variety of data visualizations, written interpretations of the data, links to relevant articles, and a "Summary & Key insights" section.

E-commerce Site: Digital Fashion

E-commerce Site (June 2023)

Python (Flask, SQAlchemy database), HTML & CSS, Stripe API
